# React Lending

Лендинг-приложение на React с базовой структурой страниц, общими компонентами и переключением языка интерфейса (EN/UA).

## Что уже реализовано

- каркас приложения на **Create React App**;
- главная страница (`Home`) как стартовая точка лендинга;
- переиспользуемые компоненты `Header` и `Footer`;
- контекст языка (`LanguageContext`) с переключателем EN/UA в шапке;
- подготовленная навигация под страницы `Gallery`, `Service`, `Contacts`.

## Стек

- React
- Reactstrap + Bootstrap
- React Router DOM
- React Helmet Async
- SCSS Modules

## Быстрый старт

```bash
npm install
npm start
```

Приложение запустится по адресу: [http://localhost:3000](http://localhost:3000).

## Доступные команды

- `npm start` — запуск в режиме разработки;
- `npm run build` — production-сборка;
- `npm test` — запуск тестов.

## Текущий статус

Проект находится в стадии развития: часть страниц уже заведена в структуре, но их контент ещё предстоит заполнить.
